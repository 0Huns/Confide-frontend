## 아키텍쳐

### 📢&ensp;파일 구조 트리

```bash
📦src
 ┣ 📂assets // 이미지 파일
 ┣ 📂component
 ┃ ┗ ...  // 프로젝트 내 컴포넌트 파일 관리
 ┣ 📂hooks
 ┃ ┗ ...  // api 관련 hooks 파일 
 ┣ 📂pages
 ┃ ┣ 📜App.js
 ┃ ┣ 📜Auth.js
 ┃ ┣ 📜Intro.js
 ┃ ┣ 📜Main.js
 ┃ ┗ 📜User.js
 ┣ 📂services
 ┃ ┗ ...  // auth 관련 api 파일
 ┣ 📂store
 ┃ ┣ 📂module
 ┃ ┗ ...  // 리덕스 모듈 파일
 ┃ ┗ 📜index.js
 ┣ 📂styles
 ┃ ┗ 📜App.css
 ┣ 📜App.test.js
 ┣ 📜index.css
 ┣ 📜index.js
 ┗ 📜setupTests.js
```
